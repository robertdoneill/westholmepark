Westholme Park为UCLA、UCLA扩展学院和圣莫尼卡学院的女生提供服务。我们的物业有研究生和本科生混合居住，并提供私人和共享卧室。

## 入门指南

要在Westholme Park预订住宿，您需要提供以下文件：护照，美国签证或在美国居住/工作的学校/工作授权，以及资金证明。

资金证明可以是学生助学金、学生贷款、在美国工作的工资、录取通知书，或银行账户显示有足够的资金支付住宿费用。

## 提交申请

您可以通过我们的在线表格[这里](https://tally.so/r/mV86WJ)提交申请，或发送电子邮件至 join@westholmepark.com。
